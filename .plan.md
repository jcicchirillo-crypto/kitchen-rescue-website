# Safely Remove Duplicate CSS Rules

## Strategy

Instead of just deleting the old rules, we need to:
1. Add any missing properties from old rules to the new rules
2. Then delete the old rules

## Changes Required:

### 1. Update NEW `.hero` rule (line 1215) to include missing properties

**Current:**
```css
.hero {
    max-width: 1080px;
    margin: 0 auto 32px;
    padding: 48px 36px;
    border-radius: 18px;
    background: linear-gradient(150deg, #141414 0%, #030303 70%);
    border: 1px solid rgba(255, 255, 255, 0.06);
    box-shadow: 0 35px 65px rgba(0, 0, 0, 0.55);
}
```

**Update to:**
```css
.hero {
    position: relative;
    overflow: hidden;
    max-width: 1080px;
    margin: 0 auto 32px;
    padding: 48px 36px;
    border-radius: 18px;
    background: linear-gradient(150deg, #141414 0%, #030303 70%);
    border: 1px solid rgba(255, 255, 255, 0.06);
    box-shadow: 0 35px 65px rgba(0, 0, 0, 0.55);
    color: #ffffff;
}
```

### 2. Delete OLD `.hero` rule (lines 364-369)
### 3. Delete OLD `.spec` rule (lines 680-682)
### 4. Delete OLD `.gallery` rule (lines 746-749)

This ensures we don't lose any important properties while cleaning up the duplicates.
