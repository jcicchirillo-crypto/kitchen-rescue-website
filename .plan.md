# Fix PDF Generation on Vercel

## Issues Identified
- PDF generation silently fails due to Chromium size/timeout issues on Vercel
- Need smaller Chromium package (chromium-min)
- Need Node 20 runtime configuration
- Need debug logs to verify PDF generation

## Implementation Steps

### 1. Swap Chromium Package
- Uninstall @sparticuz/chromium
- Install @sparticuz/chromium-min

### 2. Update PDF Builder
- Change import to use @sparticuz/chromium-min
- Add debug logs for executable path and PDF buffer size
- Store executable path in variable for logging

### 3. Configure Vercel
- Create/update vercel.json with Node 20 runtime
- Set memory to 1024MB
- Set maxDuration to 30 seconds

## Files to Modify
1. package.json (via npm commands)
2. api/lib/buildClientQuotePdf.js
3. vercel.json (create or update)
