# Fix Gallery Section Alignment

## Issue

The "See the pod inside and out" (Gallery) section has different left padding than the Spec section (where Insurance support box is):

- **Gallery**: `padding: 32px 28px 36px` → 28px left padding
- **Spec**: `padding: 32px 36px 36px` → 36px left padding

This causes misalignment between sections.

## Solution

Update the Gallery section padding to match all other sections at 36px horizontal padding.

**Change in `styles.css` (line 1252):**

**From:**
```css
.gallery {
    max-width: 1080px;
    margin: 0 auto 28px;
    padding: 32px 28px 36px;
    border-radius: 18px;
    background: #ffffff;
    border: 1px solid rgba(17, 24, 39, 0.08);
    box-shadow: 0 30px 60px rgba(15, 15, 15, 0.3);
    color: #111827;
}
```

**To:**
```css
.gallery {
    max-width: 1080px;
    margin: 0 auto 28px;
    padding: 32px 36px 36px;
    border-radius: 18px;
    background: #ffffff;
    border: 1px solid rgba(17, 24, 39, 0.08);
    box-shadow: 0 30px 60px rgba(15, 15, 15, 0.3);
    color: #111827;
}
```

This will make the Gallery section align perfectly with the Spec section (Insurance support box) and all other content sections.
